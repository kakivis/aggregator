# READ ME

This is a simple application that fetches contact info from a paginated API and returns the full list.

## Requirements
- Java 17
- SpringBoot
- Lombok
- Maven


## Compile

`mvn clean compile`

## Run tests

`mvn verify`

## Run

`mvn spring-boot:run`

The route implemented is a GET endpoint and should be under `http://localhost:8080/contacts` .
